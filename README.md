📑 python_deep 프로젝트
1. 소개글
이 저장소는 딥러닝의 가장 기초적인 예제인 Iris(붓꽃) 데이터셋과 Diabetes(당뇨병) 데이터셋을 활용한 학습 과정을 담고 있습니다. 데이터 전처리부터 모델 설계, 그리고 학습된 모델을 통해 원하는 결과값을 직접 테스트해 보는 전체 파이프라인을 구축하는 것을 목표로 합니다.

2. 기술스택
Language: Python (Pandas, Numpy)

Library: TensorFlow, Keras

Tools: Git, GitHub

3. 얻고자 하는 것
기술적 성장: 딥러닝 모델의 전처리부터 학습, 테스트까지의 전 과정을 마스터하여 실무 역량을 갖추고자 합니다.
전문성 확보: 정보처리기사 자격 취득과 꾸준한 블로그 기록(T-story)을 통해 신뢰받는 개발자로 거듭나는 것입니다.
